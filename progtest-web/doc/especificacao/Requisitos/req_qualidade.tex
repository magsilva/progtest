\subsection{Funcionalidade}
\subsection{Usabilidade}

\subsection{Confiabilidade}

O sistema deve ter capacidade para recuperar os dados perdidos da
última operação que realizou em caso de falha.

\subsection{Eficiência}

O tempo de processamento de uma operação de avaliação não deve
exceder 10 segundos.


\subsection{Manutenibilidade}

O sistema deve ser de fácil manutenção.

\subsection{Portabilidade}

O sistema deve ser portável.
